Class for building and validating forms

This class allows for building forms using J\-S\-O\-N. Fields are automatically validated based on Rules allowing for easy server-\/side validation. The markup closely resembles jquery validation plugin so you can use one stylesheet for both client-\/ and server-\/side validation.\hypertarget{nav1_usage}{}\section{Usage}\label{nav1_usage}
